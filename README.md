# 🤖 Code Reviewer AI

Um assistente inteligente de revisão de código desenvolvido em Python utilizando a API da OpenAI.

O objetivo do projeto é analisar códigos enviados pelo usuário, identificar automaticamente a linguagem de programação, encontrar possíveis erros, sugerir melhorias e fornecer uma avaliação da qualidade do código seguindo boas práticas de desenvolvimento.

---

## 📌 Sobre o Projeto

O **Code Reviewer AI** funciona como um revisor automático de código.

O usuário envia um trecho de código ou uma dúvida relacionada à programação, e o sistema utiliza inteligência artificial para:

- Identificar a linguagem utilizada.
- Analisar a estrutura do código.
- Encontrar erros de sintaxe.
- Detectar possíveis bugs.
- Avaliar organização e legibilidade.
- Sugerir melhorias.
- Explicar correções necessárias.
- Avaliar a qualidade do código com uma nota de 0 a 10.

---

## 🚀 Funcionalidades

✅ Identificação automática da linguagem de programação.

✅ Revisão de código utilizando inteligência artificial.

✅ Análise de:

- Sintaxe.
- Indentação.
- Variáveis.
- Funções.
- Estruturas condicionais.
- Laços de repetição.
- Organização.
- Segurança.
- Eficiência.

✅ Explicação detalhada dos erros encontrados.

✅ Sugestão de código corrigido.

✅ Avaliação final da qualidade do código.

✅ Interface via terminal (CLI).
